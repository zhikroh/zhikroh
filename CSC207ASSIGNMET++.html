MEANING OF HTML

HTML is a markup language for describing web documents (web pages).

  1. HTML stands for Hyper Text Markup Language

  2. A markup language is a set of markup tags

  3. HTML documents are described by HTML tags

  4. Each HTML tag describes different document content

Structure of HTML

Web Documents contains the following listed below

  • The DOCTYPE declaration defines the document type to be HTML.The declaration helps the browser to display a web page correctly. There are different document types on the web. To display a document correctly, the browser must know both type and version. The doctype declaration is not case sensitive. All cases are acceptable

  • The text between html tag describes an HTML document

  • The text between head tag provides information about the document

  • The text or content between title tag provides a title for the document

  • The text between body tag describes the visible page content. Only the body area (the white area) is displayed by the browser.

  • The text between header tag describes a heading

  • The text between paragraph tag describes a paragraph

HTML Tags


HTML tags are keywords (tag names) surrounded by angle brackets
  • HTML tags normally come in pairs like paragraph tag

  • The first tag in a pair is the start tag, the second tag is the end tag

  • The end tag is written like the start tag, but with a slash before the tag name

  • The start tag is called opening tag while end tag is called closing tag

    • Purpose of a Browser in Html


    The purpose of a web browser (Chrome, IE, Firefox) is to read HTML documents and display them. The browser does not display the HTML tags, but uses them to determine how to display the document.

    HTML Attributes

    • Attributes provide additional information about an element

    • Attributes are always specified in the start tag

    • Attributes come in name/value pairs like: name="value"


    Difference between Client side and server side

    Client is a system with browser and internet connectivity while Server is a system or computer that provides services to other users
